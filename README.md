# Portfolio Website – Pitman Training Final Assignment

This repository contains my final project for the Pitman Training HTML & CSS course.  
The project is a basic portfolio website created using HTML and CSS, demonstrating the skills learned during the course.

## Project Overview

The website includes:

• A responsive layout built with HTML and CSS  
• Navigation links to different sections  
• An "About Me" section  
• A Projects section  
• A Contact section  
• A profile image  
• Clean and structured code

## Technologies Used

• HTML5  
• CSS3  
• Git & GitHub  
• GitHub Pages for deployment
